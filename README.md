Structured dataset of open banking regulatory frameworks across major markets. Covers PSD2 (EU), CDR (Australia), Open Banking Standard (UK), CFPB 1033 (US), and equivalent frameworks in Canada and Southeast Asia.

Includes: API requirements, data sharing mandates, consent rules, and compliance timelines.

**Format:** JSON, CSV, Markdown
